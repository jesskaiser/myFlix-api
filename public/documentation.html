Introduction

The API is the server-side component of a “movies” web application. The web application provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

URL endpoints and request methods

Business Logic / Request URL HTTP Method Request Body Data Format Response Body Data Format
Add a user /users POST A JSON object holding data about the user to add, structured like:
{
username: "",
password: "",
email: ""
}		 A JSON object holding data about the user that was added, including an ID:
{
id: "",
username: "",
password: "",
email: "",
Favorite Movies: []
}
Change a user's username /users/[Username] PUT None A text message indicating the username was updated
Return data about a user /users/[ID] GET None a JSON object holding data about the user, structured like:
{
id: "",
username: "",
password: "",
email: "",
Favorite Movies: []
}
Add a movie to the user's favorites /users/[ID]/[Title] POST None Text message indicating the movie has been added to the user's list
Remove a movie from the user's favorites by title /users/[ID]/[Title] DELETE None Text message indicating the movie has been removed from the user's list
Delete a user by ID /users/[ID] DELETE None Text showing that the user email has been removed
Return a list of all movies of the user /movies GET None A JSON object holding data about all the movies, structured like:
{
id: "",
username: "",
Favorite Movies: []
}
Return data about single movie by title /movies/[Movie] GET None A JSON object holding data about a single movie, structured like:
{
title: "",
imageURL: "",
genres: [],
directors: [],
featured: y/n,
description: ""
}
Return data about a genre by name /movies/genres/[Genre] GET None A JSON object holding data about a specific genre
Return data about a director by name /movies/directors/[Director] GET None A JSON object holding data about a specific director. structured like:
{
name:"",
bio:"",
birth:""
}
